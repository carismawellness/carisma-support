`{rep_rankings_table}` is generated dynamically. Each row follows this format:

```
{rank}  {rep_name}       {rep_score}   {rep_trend}   {rep_convos}  {rep_strongest}   {rep_weakest}
```

```
5  Luke             58/100  -3      5       Greeting          Close
6  Elena            52/100  -7      4       Follow-Up         Discovery
```

### Dimension Heatmap Variables

`{dimension_heatmap_rows}` is generated dynamically. Each row:

```
{dimension_name}:            {dim_avg}    {dim_prev}   {dim_arrow}   {dim_below_5_count}
```

"Reps Below 5" counts how many reps averaged below 5/10 in that dimension this week. This highlights dimensions where multiple reps are struggling, not just one.

### Attention Reps Section

`{attention_reps_section}` is generated dynamically for each rep meeting the criteria. Format per rep:

```
{rep_name}: {rep_score}/100 ({rep_trend})
  Consecutive weeks below 60: {weeks_below_60}
  Weakest dimensions: {dim_1} ({score}), {dim_2} ({score})
  Pattern: {recurring_issue}
  Recommended action: {recommended_action}
```

`{attention_reps_empty_note}` is shown only when no reps need attention:
```
No reps scored below 60 or showed significant decline this week. The team is performing well.
```

### Attention Criteria

A rep appears in the attention section if any of the following are true:
- Average composite score below 60 this week
- Score declined by 8+ points vs previous week
- Score declined for 3 consecutive weeks
- Any domain accuracy failure (factual error about treatments, pricing, or protocols)

## Selection Logic

### Rep Rankings
Sort all reps by average composite score (descending). Include all reps who had at least 1 scored conversation.

### Dimension Heatmap
Sort dimensions by team average (ascending, weakest first). The "Reps Below 5" column makes it easy to see whether a low average is caused by one outlier or a team-wide issue.

### Top Performers
- **Most improved:** Rank by absolute point increase vs previous week. Minimum +2 points to qualify. If fewer than 2 qualify, show only those who do.
- **Highest scorer:** Simple rank by average composite score.

### Training Topics
Derived from team-wide patterns:
1. The weakest team dimension (if 3+ reps are below target)
2. The most common recurring improvement area across all individual weekly reports
3. Any dimension that declined team-wide vs previous week

### Brand Comparison
Only include brands that had 3+ conversations scored that week. If a brand had fewer than 3, show "Insufficient data" instead of scores.

## Delivery Rules

1. Sent every Monday at 8:00 AM Malta time, alongside individual weekly summaries.
2. Delivered via email to the manager.
3. A condensed version (team overview + attention reps + brand comparison only) is also available for WhatsApp if the manager prefers mobile delivery.
4. If the team average drops below 60 or declines for 2+ consecutive weeks, the report subject line is prefixed with "[ACTION NEEDED]".
5. If a rep appears in the attention section for 3+ consecutive weeks, escalate by adding a note: "This rep has been flagged for 3 consecutive weeks. Consider scheduling a 1-on-1 coaching session."
