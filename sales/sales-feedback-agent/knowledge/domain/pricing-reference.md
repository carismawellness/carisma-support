## How to Use This File

When scoring a conversation, check any quoted price against this reference. Pricing errors fall into three categories:

| Severity | Description | Example |
|----------|-------------|---------|
| **Critical** | Wrong price quoted (off by more than 10%) | Saying Botox 50u is EUR 199 when it is EUR 249-299 |
| **Major** | Quoting member price to non-member (or vice versa) | Giving EUR 224 rate without confirming membership |
| **Minor** | Rounding correctly but imprecisely | Saying "around EUR 250" for a EUR 249 treatment |

## Pricing Verification Checklist (For Scoring)

When reviewing a conversation for pricing accuracy, check:

- [ ] Was the correct member/regular rate quoted?
- [ ] Was the price within the Low-High range for the product brand?
- [ ] Was the duration correctly stated?
- [ ] Were multi-session package savings mentioned when relevant?
- [ ] Was the 3-tier framework used (budget question, 3 options, reaffirm tier 2)?
- [ ] Was pricing presented as value (not just cost)?
- [ ] Were any unauthorized discounts offered?
- [ ] Were promotional offers correctly described (inclusions, price, limitations)?
- [ ] Was member pricing only given to confirmed members?
- [ ] Were internal costs or margins accidentally disclosed?
