# Reception — Hospitality Program Knowledge

> Source: Astra Culture Hospitality Program / Reception folder

## Overview

The reception team is the first point of contact for every patient. Their role combines hospitality excellence with sales fundamentals — warm greeting, efficient information gathering, and seamless handoff to consultation.

## Key Principles (from Steps of Service Guide)

### Initial Contact (Phone)
- Answer within 3 rings
- Use a warm, welcoming tone
- Avoid giving pricing over the phone — redirect to consultation value
- Conduct light discovery: "What brought you to us today?"
- Goal: book the consultation, not sell the treatment

### Arrival and Check-In
- Greet every patient by name
- Offer refreshments immediately
- Welcome tour for first-time visitors
- **5ft/10ft Rule:** At 10 feet, make eye contact and smile. At 5 feet, greet verbally.
- Credential the provider: "You'll be seeing Dr. [Name], who specializes in..."
- Have patient notes ready before arrival

### Pre-Arrival Preparation
- Send confirmation messages (text preferred)
- Include address and parking details
- Review patient notes and history before they arrive
- Prepare any intake forms needed

## Verbiage Standards (from Verbiage Swap Guide)

Reception must use hospitality-grade language at all times:

| Never Say | Always Say |
|-----------|-----------|
| "No problem" | "My pleasure" / "Absolutely" |
| "Hold on" | "Allow me one moment" |
| "I don't know" | "Let me find out for you" |
| "You can't..." | "What I can do for you is..." |
| "Cost" / "Price" | "Investment" / "Value" |
| "Procedure" | "Treatment" / "Experience" |
| "Expensive" | "Premium" / "Elevated" |
| "Cheap" | "Accessible" / "Entry-level" |
| "Pain" | "Discomfort" / "Sensation" |
| "Problem" | "Challenge" / "Concern" |
| "Contract" | "Agreement" / "Membership" |
| "Waiting room" | "Lounge" / "Reception area" |
| "Buy" | "Invest in" / "Begin" |
| "Deal" | "Package" / "Program" |
| "Sell" | "Recommend" / "Suggest" |

## Communication Protocols

### AOA (Audible Order Acknowledgement)
When a team member calls out a task, the response is immediate:
- "AOA — on it" (I heard you, I'm handling it)
- "No Hands" (I'm currently unavailable)

Reception-specific AOA scenarios:
- **Phone coverage:** "Can someone grab the phone?" → "AOA — on it"
- **Patient walk-back:** "Ready to bring [patient] back" → "AOA — walking them now"
- **Checkout handoff:** "[Patient] is ready for checkout" → "AOA — got it"
