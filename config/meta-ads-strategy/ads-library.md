# Competitor Ad Library Research Sources
# Carisma Wellness Group — Meta Ads

---

## Purpose

This file defines which competitor brands to research in Meta's Ad Library when building new campaigns for each Carisma brand. Use these sources to:

1. Audit what hooks, offers, and formats competitors are running
2. Identify gaps Carisma can exploit
3. Borrow inspiration from high-volume advertisers in the wellness/aesthetics vertical (internationally, not just Malta)
4. Understand seasonal offer timing and pricing benchmarks

**How to use:** When briefing a new campaign, search these brand names in Meta Ad Library (ads.facebook.com/ads/library). Filter by country (Malta + UK + US) and category (All Ads). Screenshot and save top-performing ads to `$AGENT_HOME/competitor-creative/`.

## Research Protocol

When conducting Ad Library research:

1. **Search brand name** in Meta Ad Library — set to "All" regions first, then filter to Malta + UK
2. **Filter by active ads only** for current competitive intelligence
3. **Sort by impressions** (high impression ads = high spend = likely performers)
4. **Screenshot 3–5 top ads** per brand per research session
5. **Note:** hook type, offer structure, CTA, creative format, any seasonal angle
6. **Save to:** `$AGENT_HOME/competitor-creative/[brand]/[date]-[competitor-name].png`
7. **Log findings in:** `$AGENT_HOME/competitor-creative/research-log.md`

Report findings to Meta Strategist with: what competitors are running, which angles are saturated, and which gaps Carisma could exploit.
