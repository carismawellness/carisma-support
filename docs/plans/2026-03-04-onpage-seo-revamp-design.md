### What's Missing from Your Knowledge (Critical 2026 Additions)

#### 1. AI Search Engine Optimization (AEO/GEO)

This is the biggest shift in SEO since mobile-first indexing. By 2026, an estimated 60%+ of searches involve AI-generated responses (Google AI Overviews, Bing Copilot, ChatGPT Search, Perplexity).

**Key principle: "Semantic Chunking"**
Structure content so each H2 section is a **self-contained, extractable mini-resource** with a **quotable lead answer** in the first 1-2 sentences. AI engines extract these chunks verbatim.

**How to optimize for AI citability:**
- Start each section with a direct, concise answer to the heading question (40-60 words)
- Follow with supporting details, examples, and evidence
- Use structured formats: numbered lists, comparison tables, step-by-step guides
- Include specific data points (prices, durations, ratings) -- AI engines love citing specifics
- Write in a neutral, authoritative tone that AI engines trust enough to quote

**Example (bad):**
```
## Our Massages
We offer various massage treatments at our locations across Malta...
```

**Example (good -- AI-citable):**
```
## What Types of Massage Are Available in Malta?
Carisma Spa offers 12 massage types across 8 locations in Malta,
ranging from EUR39 for a 30-minute express massage to EUR120 for
a 90-minute couples package. The most popular treatments include
Thai massage, deep tissue, and hot stone therapy.
```

The second version is what AI engines will quote. It's specific, factual, and self-contained.

#### 2. E-E-A-T Signals (Experience, Expertise, Authoritativeness, Trustworthiness)

Google's 2024 Helpful Content Update and 2025 Quality Rater Guidelines make E-E-A-T signals a direct ranking factor for health/wellness content (YMYL -- Your Money Your Life).

**On-page E-E-A-T signals for Carisma:**
- **Experience:** First-person practitioner content. "Our senior therapist Maria explains..." Add therapist bios with qualifications and years of experience.
- **Expertise:** Mention certifications (CIBTAC, ITEC, CIDESCO). Link to professional bodies. Include "About our therapists" sections.
- **Authoritativeness:** Display awards prominently ("Award-Winning Spa"). Show Google review ratings on-page. Link to press mentions/media features.
- **Trustworthiness:** SSL (you have this). Clear pricing. Real business address/phone visible on every page. Privacy policy. Terms & conditions. Association memberships. Insurance details.

#### 3. Passage-Based Ranking & Featured Snippets

Since 2021, Google can rank individual **passages** within a page, not just the whole page. In 2026, this is amplified by AI Overviews pulling specific passages.

**What this means for Carisma:**
- A single well-written paragraph can rank for a keyword even if the page targets a different primary keyword
- Every H2 section should be optimized as a standalone answer
- Use the "inverted pyramid" style: answer first, details second, context third

#### 4. Core Web Vitals / INP (Interaction to Next Paint)

In March 2024, Google replaced FID with INP (Interaction to Next Paint). This measures how quickly your page responds to user interactions (clicks, taps, keyboard input).

**For Wix sites:**
- Lazy-load images below the fold
- Use WebP/AVIF image formats (Wix supports this natively)
- Minimize custom JavaScript
- Keep images under 150KB (you already noted this -- still valid)
- Use Wix's built-in performance tools

#### 5. Internal Linking: Hub-and-Spoke Model

Each money page is a **hub**. Related pages (blog posts, location pages, sub-service pages) are **spokes** that link back to the hub with keyword-rich anchor text.

```
                    Homepage (spa malta hub)
                   /    |     |      \
          /massages  /spa-day  /laser  /facials  (service hubs)
         /  |  \      |   \      |
   /thai  /deep  /couples  ...  /blog-posts
```

**Rules:**
- Every spoke page links to its hub with descriptive anchor text (not "click here")
- Hubs link to each other (cross-service promotion)
- Blog posts link to the most relevant service page
- 5-10 internal links per page minimum, 15-20 for pillar pages
- Vary anchor text (don't always use exact match keyword)

#### 6. Multi-Language Hreflang

Given Carisma targets tourists from multiple countries, implementing hreflang tags for at least English, Maltese, Italian, French, German, Spanish would capture tourist search intent in their native language. Competitor MyGuideMalta has 15 languages with hreflang -- this is a significant competitive gap.

#### 7. Review Integration with Schema

Neither competitor has AggregateRating schema linking to real reviews. Carisma has excellent Google review scores (4.8/5 with 970+ reviews at Hugo's location). Embedding these on-page with AggregateRating schema can trigger star ratings in SERPs.

#### 8. SpeakableSpecification for Voice Search

New in 2025-2026: `SpeakableSpecification` schema tells voice assistants (Google Assistant, Alexa, Siri) which sections of your page to read aloud. Add this to your most concise, quotable content blocks.

## Part 2: Master On-Page SEO Checklist (2026)

Every page in the revamp must satisfy these requirements:

### Meta Tags
- [ ] Title tag: 50-60 chars. Primary keyword in first 30 chars. One variation allowed. Compelling/clickable.
- [ ] Meta description: 140-155 chars. Primary keyword once (not first word). Include CTA. Use CAPITALS for key words. Include a cliffhanger or value prop.
- [ ] Canonical tag: Self-referencing.
- [ ] OG title + OG description + OG image (1200x630px minimum).
- [ ] Twitter card: summary_large_image.

### Heading Structure
- [ ] Single H1 per page. Contains primary keyword (partial match OK).
- [ ] H2s: 5-8 per page. Mix of keyword variations + question formats (who, what, where, how much, for who).
- [ ] H3s: Under each H2 as needed. Use semantically related terms, not exact keyword copies.
- [ ] No heading tags on decorative/footer text (currently broken on all 4 pages).
- [ ] 60% of content falls below the H1 tag.

### Content Structure (Semantic Chunking for AI)
- [ ] Each H2 section starts with a direct answer (40-60 words) followed by supporting detail.
- [ ] Primary keyword in first paragraph and last paragraph.
- [ ] 2-3 bold phrases per page (key benefits/differentiators, not just keywords).
- [ ] Bullet lists for features, services, benefits.
- [ ] Comparison table where relevant (pricing, packages).
- [ ] Minimum word count: 1,500-2,500 for service pages (match top 10 ranking pages).
- [ ] Reading level: Grade 8-10 (Flesch-Kincaid).

### FAQ Section
- [ ] 5-8 FAQs per page, framed as questions people actually ask.
- [ ] Each answer starts with a direct response (1-2 sentences) then elaborates.
- [ ] Keyword included naturally in both questions and answers.
- [ ] FAQPage schema markup applied.

### Schema Markup (JSON-LD)
- [ ] HealthAndBeautyBusiness or DaySpa schema (not generic LocalBusiness).
- [ ] Service schema for each treatment/service with Offer pricing.
- [ ] FAQPage schema for FAQ sections.
- [ ] AggregateRating schema with real Google review data.
- [ ] BreadcrumbList schema.
- [ ] SpeakableSpecification on key content blocks.

- [ ] At least 1 hero image per page with primary keyword in alt and filename.

### Internal Linking

- [ ] Link to all related service pages (hub-and-spoke).
- [ ] Varied anchor text (mix of exact match, partial match, and natural phrases).
- [ ] Every page links back to homepage.

- [ ] Remove self-referential anchor links (currently polluting all 4 pages).

### E-E-A-T Signals

- [ ] Google review ratings embedded on-page.
- [ ] Real business NAP (Name, Address, Phone) visible -- not just in footer.

- [ ] Insurance details (if applicable).

### Technical
