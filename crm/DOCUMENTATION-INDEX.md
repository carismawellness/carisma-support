## Training Path

### Path A: New Agent (5-Day Onboarding)

```
DAY 1 (2-3 hours)
├─ Morning: Technical setup
│  └─ Install Claude Code, clone repo, open in brand folder
├─ Afternoon: Documentation review
│  └─ Read QUICK-START-GUIDE → AGENT-WORKFLOW-SETUP → CLAUDE.md
└─ Success: Can open Claude Code, understand basic process

DAY 2 (2-3 hours)
├─ Morning: 5 practice runs (with team lead)
│  └─ Run /crmrespond with sample customer messages
├─ Afternoon: 5 independent runs (team lead reviews)
│  └─ You lead workflow, team lead supervises
└─ Success: Scored 85+ on 10 responses, understand validation

DAY 3 (2-3 hours)
├─ Morning: 5 live responses (team lead watching)
│  └─ Real customers, you generate, team lead approves
├─ Afternoon: 5-10 live responses (team lead spot-checks)
│  └─ You lead, team lead reviews before sending
└─ Success: 10-15 real messages handled with oversight

DAY 4 (2-3 hours)
├─ Morning: 5 independent messages (spot checks)
│  └─ You handle independently, team lead reviews 2 of them
├─ Afternoon: 5-10 messages focusing on speed
│  └─ Track time, aim for 1-1.5 minutes per message
└─ Success: 12-15 messages/day, 87+ average score

DAY 5 (4-hour shift + 4-hour shift)
├─ Morning shift: Full 4 hours independently
│  └─ First full shift, 4-5 messages per hour
├─ Afternoon shift: Full 4 hours with minimal oversight
│  └─ Continue handling messages independently
└─ Success: 15-20+ messages/day, 88+ average score, ready for independent work

RESULT: Agent is ready for independent operation
```

## Quality Standards

All three documents emphasize:

1. **Never send below 85/100 validation score**
   - Below 85 means brand rules were violated
   - Exception: 70-84 is acceptable to send (but consider revising)

2. **Always use correct brand signature**
   - Aesthetics: "Beautifully yours, Sarah"
   - Spa: "Peacefully, Sarah"
   - Slimming: "With you every step, Katya"

3. **Copy exact customer message (don't edit)**
   - Preserve tone, emoji, typos, capitalization
   - System uses exact message to match energy

4. **When in doubt, rerun /crmrespond**
   - Getting <85? Rerun often improves it
   - Most issues resolve on 1-2 reruns

5. **Ask for help when needed**
   - Stuck below 70 after 3 reruns? Ask team lead
   - Unsure if response is safe? Ask team lead
   - Unclear brand rules? Ask team lead
