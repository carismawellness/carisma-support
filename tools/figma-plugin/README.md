# Ad Script Importer - Figma Plugin

Automatically import ad scripts generated by Claude into your Figma design files.

## Overview

This plugin works with `write_scripts_to_figma.py` to create a seamless workflow:

1. **Generate scripts** using Claude (Workflow 04)
2. **Export to JSON** using the Python tool
3. **Import to Figma** using this plugin

---

## Installation

### Step 1: Install Plugin in Figma

1. **Open Figma Desktop App** (plugins require desktop app, not browser)

2. **Go to Plugins:**
   - Menu → Plugins → Development → Import plugin from manifest

3. **Select manifest.json:**
   - Navigate to: `tools/figma-plugin/manifest.json`
   - Click "Open"

4. **Plugin Installed!**
   - The "Ad Script Importer" plugin is now available in your Figma desktop app

## Setup Your Figma File

### Text Layer Naming Convention

For the plugin to work, name your text layers using this pattern:

```
Script_1
Script_2
Script_3
Script_4
Script_5
...
```

**Example Figma Structure:**

```
📄 Page: Ad Scripts
  ├─ 🖼️ Frame: Medical Weight Loss Carousel
  │   ├─ 📝 Script_1  ← Hook text layer
  │   ├─ 📝 Script_2  ← Problem text layer
  │   ├─ 📝 Script_3  ← Solution text layer
  │   ├─ 📝 Script_4  ← Results text layer
  │   └─ 📝 Script_5  ← CTA text layer
  │
  ├─ 🖼️ Frame: CoolSculpting Static Ad
  │   ├─ 📝 Script_1  ← Headline
  │   ├─ 📝 Script_2  ← Body copy
  │   └─ 📝 Script_3  ← CTA
```

**Tips:**
- Use exactly `Script_` prefix (case-sensitive)
- Number sequentially (Script_1, Script_2, Script_3...)
- Plugin matches by name, so naming must be exact
- You can have multiple artboards with duplicate names (plugin updates all matches)

## Usage Workflow

### Step 1: Generate Scripts

Generate ad scripts using Claude (Workflow 04):

```python
# Workflow 04 generates scripts like:
[
  {
    "name": "Medical Weight Loss - Hook",
    "content": "Tried every diet for 10 years? Time to talk to an actual doctor."
  },
  {
    "name": "Medical Weight Loss - Problem",
    "content": "Nothing worked until I met Dr. [Name] at Carisma Slimming"
  },
  ...
]
```

Save to: `.tmp/scripts/medical_weight_loss_scripts.json`

---

### Step 2: Export for Figma

Run the Python tool to generate Figma-compatible JSON:

```bash
python tools/write_scripts_to_figma.py \
  --file "YOUR_FIGMA_FILE_KEY" \
  --scripts ".tmp/scripts/medical_weight_loss_scripts.json" \
  --output ".tmp/scripts/figma_import.json"
```

**How to get your Figma file key:**
- Open your Figma file in browser
- URL format: `https://www.figma.com/file/ABC123XYZ/File-Name`
- File key is `ABC123XYZ`

This generates: `.tmp/scripts/figma_import.json`

---

### Step 3: Import to Figma

1. **Open your Figma file** (must use Figma Desktop App)

2. **Run the plugin:**
   - Menu → Plugins → Development → Ad Script Importer

3. **Copy the JSON:**
   ```bash
   cat .tmp/scripts/figma_import.json | pbcopy
   ```
   (Or open the file and copy contents manually)

4. **Paste into plugin:**
   - Paste JSON into the text area
   - Click "Import Scripts"

5. **Done!**
   - Text layers named `Script_1`, `Script_2`, etc. are now updated
   - ✅ Success message shows how many scripts were imported

## Troubleshooting

### "Layer not found: Script_X"

**Problem:** Plugin can't find a text layer with that name.

**Solution:**
- Check layer naming (must be exactly `Script_1`, `Script_2`, etc.)
- Click "List Text Layers" to see current names
- Make sure you're on the correct Figma page

### "Invalid JSON"

**Problem:** The JSON format is incorrect.

**Solution:**
- Verify you copied the entire JSON file
- Check for missing brackets or quotes
- Re-run `write_scripts_to_figma.py` to regenerate

### Plugin not showing in menu

**Problem:** Plugin not installed or Figma needs restart.

**Solution:**
- Restart Figma Desktop App
- Re-import plugin manifest
- Make sure you're using Figma Desktop (not browser)

### Font loading errors

**Problem:** Text layer uses a font not loaded.

**Solution:**
- Install missing fonts
- Or change text layer to use a standard font before import

## Advanced Usage

### Custom Layer Naming Pattern

If you want to use a different naming pattern (e.g., `AdScript_` instead of `Script_`):

```bash
python tools/write_scripts_to_figma.py \
  --file "YOUR_FILE_KEY" \
  --scripts ".tmp/scripts/scripts.json" \
  --layer-pattern "AdScript_"
```

Then name your Figma layers: `AdScript_1`, `AdScript_2`, etc.

---

### Automation with Workflows

Integrate into your WAT framework:

**Workflow 04 (Script Generation)** already generates scripts as JSON.

Add this step to automatically prepare for Figma import:

```python
# At end of Workflow 04
import subprocess

subprocess.run([
    "python", "tools/write_scripts_to_figma.py",
    "--file", FIGMA_FILE_KEY,
    "--scripts", ".tmp/scripts/output.json",
    "--output", ".tmp/scripts/figma_import.json"
])

print("✅ Figma import file ready: .tmp/scripts/figma_import.json")
print("   Open Figma plugin and paste contents to import.")
```
