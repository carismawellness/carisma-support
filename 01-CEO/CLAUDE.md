# CEO Workspace

Executive workspace for the CEO of Carisma Wellness Group. Handles WhatsApp integration, message management, strategic decision-making workflows, and daily operational summaries.

---

## Purpose

This folder centralises the CEO's personal automation tools — primarily WhatsApp management and strategic communication. Everything here supports faster decision-making and streamlined executive workflows.

## Folder Structure

| Folder | Purpose |
|--------|---------|
| `knowledge/` | Reference docs — WhatsApp tools guide, business context |
| `skills/` | Executable Claude Code skills (summarize-chat, send-message) |
| `hooks/` | Triggers and templates for automated CEO workflows |
| `config/` | Configuration files and credential references |
| `tools/` | Python scripts for WhatsApp operations |
| `workflows/` | Markdown SOPs for daily routines and quick responses |
