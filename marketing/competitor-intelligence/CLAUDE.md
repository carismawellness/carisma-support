# Competitor Intelligence

Weekly competitor ad monitoring, scraping, and analysis for Carisma Wellness Group. Tracks competitor Meta Ads activity across the wellness, aesthetics, and slimming verticals in Malta.

---

## Purpose

Automated competitor surveillance — scrapes the Meta Ad Library weekly, analyses creative patterns, extracts hooks and angles, and flags new competitor activity. Feeds insights into creative strategy.

## Folder Structure

| Folder | Purpose |
|--------|---------|
| `knowledge/` | Competitor profiles, historical analysis, industry benchmarks |
| `skill/` | Agent skill definition for competitor spy automation |
| `hooks/` | Triggers and templates for competitor monitoring |
| `scheduling/` | launchd plist for weekly automated execution |
| `tools/` | Python scraping and analysis scripts |

## Key Files

- `workflow.md` — Full SOP for the weekly competitor ad spy workflow
- `strategy.md` — Competitor intelligence strategy and objectives
- `analysis-templates.md` — Templates for structuring competitor analysis

## Related Config

- `config/competitors.json` — Competitor page IDs and Ad Library search terms
