# GBP Event Post Templates

**Last updated:** 2 March 2026
**Post type:** Google Business Profile "Event" post
**Brands:** Carisma Spa | Carisma Aesthetics | Carisma Slimming
**Usage:** Select a template, fill in placeholders, set event dates in GBP, publish.

---

## About GBP Event Posts

GBP "Event" posts display with a start date, end date, and title prominently shown. They are ideal for:

- Open days and consultation events
- Seasonal promotions with fixed dates
- Workshop or wellness experience events
- Launch events for new treatments or locations

**GBP Event Fields:**
- **Title:** Event name (max 58 characters for full display)
- **Start date/time:** When the event starts
- **End date/time:** When the event ends
- **Details:** Description of the event
- **CTA button:** Book, Learn more, Sign up, Call now
- **CTA link:** URL for the event

## Template 1: Open Day / Consultation Day

**Best for:** Free consultation events, open days, meet-the-team events
**When to use:** Once per quarter per brand, or when launching new services
**Goal:** Lower the barrier to first visit by creating a dedicated, no-pressure event

### Event Fields

```
TITLE: {EVENT_TITLE}
START DATE: {DATE} at {TIME}
END DATE: {DATE} at {TIME}
CTA BUTTON: Book
CTA LINK: {BOOKING_URL}
```

### Event Description Template

```
{EVENT_NAME} at {BRAND_NAME} in {LOCATION}.

{WHAT_THE_EVENT_IS}. {WHO_IT_IS_FOR}.

What to expect:
- {ACTIVITY_1}
- {ACTIVITY_2}
- {ACTIVITY_3}

{NO_OBLIGATION_STATEMENT}. {WHAT_YOU_WILL_LEARN_OR_EXPERIENCE}.

{DATE_AND_TIME}. {CAPACITY_STATEMENT}.

{CTA_STATEMENT}. Spaces are limited.

{BRAND_NAME}, {LOCATION}
{PHONE}
```
