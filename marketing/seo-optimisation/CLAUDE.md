# SEO Optimisation

Search engine optimisation operations for Carisma Wellness Group. Encompasses Google Search Console quick-win hunting and automated Wix meta tag optimisation.

---

## Purpose

Two-pronged SEO automation: (1) GSC Quick-Win Hunter identifies high-potential keyword opportunities from Search Console data, and (2) Wix SEO Auto-Optimiser pushes optimised meta titles and descriptions to the Wix sites.

## Folder Structure

| Folder | Purpose |
|--------|---------|
| `knowledge/` | SEO best practices, algorithm updates, keyword research |
| `skills/` | Agent skill definitions — GSC Hunter and Wix SEO |
| `hooks/` | Triggers and templates for SEO workflows |
| `scheduling/` | launchd plists for automated execution |
| `tools/` | Python scripts for GSC analysis and Wix meta optimisation |
| `workflows/` | SOPs for GSC quick wins and Wix SEO optimiser |

## Key Files

- `strategy.md` — SEO strategy and objectives
- `quick-win-criteria.json` — Criteria for identifying quick-win keywords
- `wix-seo-rules.json` — Rules for meta tag optimisation
