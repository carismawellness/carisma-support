# Google Business Profile (GMB)

Google Business Profile post generation and management for all Carisma Wellness Group locations. Automated content creation using keyword-optimised templates across all three brands.

---

## Purpose

Generates SEO-optimised Google Business Profile posts for all Carisma locations. Uses keyword banks, content calendars, and brand-specific templates to maintain consistent, high-quality local search presence.

## Folder Structure

| Folder | Purpose |
|--------|---------|
| `knowledge/` | GBP best practices, local SEO research, algorithm updates |
| `skill/` | Agent skill definition for GBP post generation |
| `hooks/` | Triggers and templates for GBP workflows |
| `scheduling/` | launchd plist for automated post generation |
| `tools/` | Python scripts for generating and checking pending posts |
| `templates/` | Post templates by type — updates, offers, events |
| `keyword-banks/` | Brand-specific keyword banks for SEO optimisation |
| `post-log/` | Log of generated posts and their status |

## Key Files

- `workflow.md` — Full SOP for GBP post generation
- `strategy.md` — GBP content strategy and objectives
- `content-calendar.json` — Monthly content calendar with themes and post types
- `locations.json` — All Carisma locations and their GBP profiles
