## Linking Rules (Quick Reference)

1. **First occurrence only** — never link the same keyword or URL more than once per post
2. **Never link in H1, H2, or H3** — body text only
3. **3–6 total internal links per post** — fewer = too sparse; more = over-optimisation risk
4. **No self-linking** — if the post is about fat freezing Malta, don't link "fat freezing Malta"
5. **Natural anchor text** — the keyword IS the anchor; never "click here" or "learn more"
6. **Audit existing links first** — writer may have already placed some; don't duplicate
7. **Stay on-brand** — only link to pages on the same brand's website
